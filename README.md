

# Efficient Bus Transfer system for Disabled Project 
 
 ![EBTD_Logo](https://user-images.githubusercontent.com/74331466/118402595-86a8ca80-b6a5-11eb-9c02-97ce60149e2c.PNG)
 
### 휠체어 이용자 & 시각장애인을 위한 버스 예약 탑승 시스템         


### 주요기능

1. 버스회사
    > * 노선 관리
    > * 기사 관리
    > * 정류장 관리
    > * 고객센터

2. 관리자
    > * 회사 관리
    > * 이용자 관리
    > * 정류장 관리
    > * 고객센터

3. 이용자
    > * 예약하기
    > * 예약확인
    > * 즐겨찾기
    > * 마이페이지

4.  버스기사
    > * 기사 설정
    > * 기사 메인화면
